# andrewpassanisi.com

## Build Setup

```bash
# Switch to the correct node version
$ nvm use

# Install dependencies
$ npm i

# Development at https://localhost:3000
$ npm run dev

# Run tests
$ npm run test

# Generate static build
$ npx nuxt generate

# Serve static build at https://localhost:3000
$ npx nuxt serve

# Serve Storybook
$ npm run storybook

```
